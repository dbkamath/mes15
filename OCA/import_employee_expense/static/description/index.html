Import HR-Employee Expense and Expense Sheet from Excel Odoo Apps

Import HR Expense and Expense Sheet odoo app helps user to import multiple(Bulk) employee expenses from XLS or CSV file, also give functionality to import expense product by Name, Internal Reference or Barcode and expense employee by Name, Work Phone, Work Email.

Import Expenses Access Rights

Under user configuration, User have to enable "Import Expenses" option to import employee expenses from XLS or CSV file.

Import Employee Expense Menu

Under Expenses > Import > Import Employee Expenses menu user can import employee expenses.

On import wizard user have to upload file and select file type, also select "Product By" and "Employee By" options.

Imported Employee Expenses

User can see imported employee expenses from uploaded file.

User can see employee expense in form view with all given data from excel file.
